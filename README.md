# ToDo List API с матрицей Эйзенхауэра

**Версия 2.0.0** - Полностью переработанное API для управления задачами с использованием матрицы Эйзенхауэра и современного стека технологий.

## О проекте

Это высокопроизводительное REST API для управления задачами, которое классифицирует задачи по четырем квадрантам матрицы Эйзенхауэра:

- **Q1**: Важные и срочные задачи
- **Q2**: Важные, но не срочные задачи  
- **Q3**: Срочные, но не важные задачи
- **Q4**: Не срочные и не важные задачи

## Технологии

- **FastAPI** - современный, быстрый веб-фреймворк
- **SQLAlchemy 2.0** - ORM с асинхронной поддержкой
- **PostgreSQL** - основная база данных (через Supabase)
- **Pydantic** - валидация данных и сериализация
- **Uvicorn** - ASGI сервер
